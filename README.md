# berry-flux-diag

Main dependencies: qeschema, h5py, and pymatgen

to run example:
- clone and install in an environment that has qeschema, h5py, and pymatgen
- from the main folder run: python tests/test_BaTiO3_QE_no_spin_polarization.py


to run bfd code need:
1. polar .xml file
2. non-polar .xml file
3. path to folder with polar wavefunctions in .hdf5 format
4. path to folder with non-polar wavefunctions in .hdf5 format
5. non-polar .out file
